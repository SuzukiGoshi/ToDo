# ToDoApplication

"ToDoApplication" is a simple ToDo application.

# Usage
1.入力を行いボタンを押すことでリストにToDoを追加することができます。  
2.追加したToDoは完了したらチェックボックスを押すことで処理済みとすることができます。  
3.右側のバツを押すことでリストから削除することができます。  
4.登録状態は保存され、閉じたときの状態が表示されます。  

# Note
・空の名称は登録することができません。  
・同名はリストに追加することができません。  
・登録できる名称の長さは20文字までです。
